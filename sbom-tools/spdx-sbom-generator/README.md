# SPDX (SBOM) Generator

- Command line tool
- Supported schemas:
  - SPDX 2.2 (default format)
  - JSON
  - RDF (in progress)

Generating SBOM from project's root with spdx-sbom-generator:

SPDX format:
```
spdx-sbom-generator -o /out/diretory/
```
JSON format:
```
spdx-sbom-generator -o /out/diretory/ -f json
```
