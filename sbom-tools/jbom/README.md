# Eclipse jbom

- Available Jar plugin
- Generates runtime and static SBOMs for local and remote Java apps
- Supported schemas:
    - CycloneDX (JSON)

Generating SBOM from project's root with jbom-1.2.1.jar:
```
java -jar jbom-1.2.1.jar -d <PROJECT_DIRETORY>
```
